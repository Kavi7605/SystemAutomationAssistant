# System Automation Assistant

An AI-powered desktop and remote automation assistant built using Python, Ollama, and Agentic AI concepts.

## Project Overview

System Automation Assistant (SAA) is an intelligent automation agent capable of understanding natural language and voice commands, generating execution plans, and performing system-level tasks automatically.

Examples:

- Open Chrome
- Create a folder named Internship
- Take a screenshot and save it
- Search Python Automation on Google
- Generate documents and reports
- Execute multi-step workflows

## Tech Stack

- Python 3.13
- Ollama
- Llama 3
- Gemma 3 (4B)
- PyAutoGUI
- PyWin32
- SpeechRecognition
- Python-Docx
- ReportLab

## Architecture

The project follows an Agentic AI architecture:

User Input
→ NLP / LLM
→ Workflow Planner
→ Tool Manager
→ Automation Engine
→ System Action
